# Spacing

O espaçamento é a distância entre os elementos em uma página. Pode ser usado para criar hierarquia visual para o conteúdo e orientar o foco para determinados elementos. Informações muito densas podem ser difíceis de digerir para um usuário, portanto, certifique-se de deixar espaço suficiente nas interfaces do usuário.

export const spacingSizes = Object.keys(spacingToken);
