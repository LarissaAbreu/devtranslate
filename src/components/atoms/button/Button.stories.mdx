O componente **Button** serve para representar um botão, ou seja, uma área clicável que executará alguma ação específica.

## Tamanhos

Com a propriedade **size**, é possível configurar o componente em três tamanhos diferentes:

export const buttonSizes = [
  { size: 'l', text: 'Large' },
  { size: 'm', text: 'Medium' },
  { size: 's', text: 'Small' },
];

export const renderButtonSizes = (variant) => {
  return (
    <Grid>
      {buttonSizes.map((item, index) => {
        return (
          <View paddingRight={{ xs: 'xs' }} paddingBottom={{ xs: 'm' }} key={`button-${index - 1}`}>
            <Button variant={variant} size={item.size} onClick={() => alert('Exemplo do onClick do Button!')}>
              Button {item.text}
            </Button>
          </View>
        );
      })}
    </Grid>
  );
};

<Story name="Size">
  <Grid paddingTop={{ xs: 'xs' }} paddingBottom={{ xs: 'xs' }}>
    {renderButtonSizes('primary')}
    {renderButtonSizes('secondary')}
    {renderButtonSizes('tertiary')}
    {renderButtonSizes('ghost')}
  </Grid>
</Story>

## Ícones

Você pode adicionar ícones utilizando a propriedade **icon** ao componente Button.

export const buttonVariant = ['primary', 'secondary', 'tertiary', 'ghost'];

<Story name="Icon (Right)">
  <Grid paddingTop={{ xs: 'xs' }} paddingBottom={{ xs: 'xs' }}>
    {buttonVariant.map((item, index) => {
      return (
        <View paddingRight={{ xs: 'xs' }} key={`button-${index - 1}`}>
          <Button
            variant={item}
            size="m"
            icon={{ name: 'search' }}
            onClick={() => alert('Exemplo do onClick do Button!')}
          >
            Icon
          </Button>
        </View>
      );
    })}
  </Grid>
</Story>

Caso queira que o ícone apareça do lado esquerdo do conteúdo utilize a propriedade **reverse**.

<Story name="Icon (Left)">

            onClick={() => alert('Exemplo do onClick do Button!')}

            Icon

        </View>
      );
    })}
  </Grid>
</Story>

## Carregando

Passe o prop **loading** para mostrar seu estado de carregamento. Por padrão, o botão mostrará um spinner e deixará a largura do botão inalterada.

<Story name="Loading">
  <Grid paddingTop={{ xs: 'xs' }} paddingBottom={{ xs: 'l' }}>
    {buttonVariant.map((item, index) => {
      return (
        <View paddingRight={{ xs: 'xs' }} key={`button-${index - 1}`}>
          <Button variant={item} loading={true} onClick={() => alert('Exemplo do onClick do Button!')}>
            Loading
          </Button>
        </View>
      );
    })}
  </Grid>
</Story>
